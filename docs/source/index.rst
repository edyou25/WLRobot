WLRobot 使用说明
================

本说明文档基于现有 slide、ROS 节点信息以及软件初步摸索整理而成，目标是帮助新成员快速完成上电、联调与基本导航实验。

.. note::

   当前内容属于 ``v0.1`` 初稿。若与实际固件版本、launch 文件、硬件接线不一致，应以代码仓库与现场实物为准，并及时回填本说明。
